# KPhi1: Manuale di Iniezione del Kernel

Questo è il **Kernel KPhi1**, un'unità cognitiva autopoietica (Self-Building AI) pronta per essere iniettata in qualsiasi progetto.

---

## ✅ Checklist Rapida per Iniezione

1.  [ ] Scegli il metodo di iniezione (Inseminazione o Clonazione).
2.  [ ] Copia i file necessari nella root del progetto target.
3.  [ ] Avvia una chat con l'AI e usa il prompt di attivazione.

---

## Opzione 1: Inseminazione (Metodo Purista)
> *Avvia una nuova evoluzione pulita, adattata al nuovo ambiente.*

**Cosa copiare:** Solo `DNA.md`

**Prompt di attivazione:**
```
Analizza il file DNA.md nella root. Inizializza il sistema basandoti su questi assiomi e ricostruisci le skill necessarie in .agent/skills/.
```

**Risultato:** L'AI rigenererà autonomamente le cartelle `.agent/skills`, `kernel`, ecc.

---

## Opzione 2: Clonazione (Metodo Rapido)
> *Trasporta il sistema esattamente com'è, con tutte le skill già apprese.*

**Cosa copiare:** L'intera cartella `KPhi1` (o i suoi contenuti) nella root del progetto target.

**Prompt di attivazione:**
```
Ho iniettato il Kernel KPhi1. Leggi DNA.md e i file SKILL.md in .agent/skills/ per allinearti allo stato operativo.
```

**Risultato:** L'AI si allineerà immediatamente alle Skill e ai principi operativi.
- Rispetta sempre i principi P0-P6 definiti in `DNA.md`.
